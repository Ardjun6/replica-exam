# Daily Template

The daily template gives users a repeatable starting point for date-based notes. It should create structure without making every day feel overdesigned.

## Template sections

- Focus, tasks, notes, links, and review.
- Optional fields for mood, energy, or project only if the workspace owner wants them.
- Simple variables for date and title.

## Template rules

- Generated notes should be editable like any other page.
- Variables should fail visibly if they cannot be resolved.
- The template should stay short by default.

## Review checklist

- A daily note created from the template is valid Markdown.
- The date title matches the daily-note configuration.
- Backlinks from daily notes are indexed.
