# Relationship Graph

The relationship graph shows how pages, attachments, tags, and map nodes connect. It should help users navigate and audit their workspace, not just display a pretty network.

## Graph modes

- Global graph for the full workspace.
- Local graph around the active page.
- Filtered graph by folder, tag, property, collection, or link type.

## Interaction rules

- Clicking a node opens the page or attachment context.
- Edges should explain the relationship type when hovered or selected.
- Filters must be easy to clear so users do not lose trust in what they are seeing.

## Review checklist

- Graph data comes from the same relationship index as backlinks.
- Large workspaces degrade gracefully.
- Filters and node visibility are stored in profile settings where appropriate.
