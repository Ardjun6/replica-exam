# Hotkeys

Hotkeys make Replica.md feel fast once users settle into it. The default set should cover common writing and navigation actions without creating conflicts across platforms.

## Default shortcuts

- Open command search, quick open, search workspace, toggle preview, open backlinks, create page, and switch panes.
- Use platform conventions for modifier keys.
- Keep risky actions away from accidental single-key shortcuts.

## Customization

- Store user changes in the `.replica` profile.
- Warn when a shortcut conflicts with another command.
- Expose commands even when no hotkey is assigned.

## Review checklist

- Every shortcut maps to a command entry.
- Conflicts are visible before saving.
- Keyboard-only workflows are tested.
