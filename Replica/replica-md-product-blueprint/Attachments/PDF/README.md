# PDF Attachment Policy

PDF attachments should remain normal files while Replica.md offers useful previews and links. The policy separates viewing from editing so the first version stays focused.

## Supported behavior

- Open PDFs from links, embeds, search results, and attachment lists.
- Show page navigation, zoom, and open-in-system controls where available.
- Track PDF references in backlinks and graph relationships.

## Limits

- No PDF editing in the baseline plan.
- No remote conversion for previews.
- Password-protected or damaged PDFs should show a clear recovery state.

## Review checklist

- PDF links are indexed as attachments.
- Viewer behavior matches the PDF viewer note.
- Publishing handles included PDFs deliberately.
