# Guide - Setup

The setup guide should help someone open the workspace, understand the folder structure, and start reviewing the blueprint without guessing what each directory means.

## First steps

- Unzip the package into a normal folder.
- Open the README and follow the recommended reading path.
- Review `.replica` only as configuration examples, not as hidden content.

## Folder tour

- Strategy, executive summary, architecture, core system, editor, knowledge layer, distribution, extensions, interface design, quality, reference, guide, sample notes, templates, attachments, collections, maps, and exports.
- Each section links back to the hub through backlinks.

## Review checklist

- A newcomer can locate the main product notes.
- Configuration files are explained clearly.
- Setup does not depend on a specific machine path.
