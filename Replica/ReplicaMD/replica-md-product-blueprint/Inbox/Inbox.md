# Inbox

The inbox is a lightweight capture area for notes that are not ready to be filed. It should be useful without becoming a permanent dumping ground.

## Capture rules

- Use short entries for ideas, links, questions, and unfinished notes.
- Process entries into pages, tasks, collections, or daily notes during review.
- Keep enough context that an item still makes sense later.

## Review habits

- Link processed items to the page they became.
- Delete or archive stale capture notes deliberately.
- Use search to find unprocessed inbox items.

## Review checklist

- Inbox items can be promoted into normal pages.
- Capture does not require a special database.
- Links from inbox notes appear in backlinks.
